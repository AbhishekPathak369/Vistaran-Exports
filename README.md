# Vistaran Exports

A responsive business website developed using React.js for showcasing export and trading services, product categories, testimonials, and company information. The project focuses on clean navigation, dynamic content rendering, and an organized user experience for visitors.

## Live Demo

* Live Website: [https://vistaran-exports.vercel.app/](https://vistaran-exports.vercel.app/)
* GitHub Repository: [https://github.com/AbhishekPathak369/Vistaran-Exports.git](https://github.com/AbhishekPathak369/Vistaran-Exports.git)

---

# Features

* Responsive multi-page business website
* Dynamic banner carousel with auto-slide functionality
* Product category section with carousel navigation
* Testimonials section with dynamic rendering
* Contact form integration using Web3Forms
* Navigation using React Router
* Reusable React components
* Dynamic content rendering using JavaScript map function
* State management using React Hooks

# Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/AbhishekPathak369/Vistaran-Exports.git
```

2. Navigate to the project folder

```bash
cd Vistaran-Exports
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
